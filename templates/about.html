Kicksaver

Find and save Kickstarter projects ending soon.

E.C. Mendenhall made Kicksaver to help out struggling Kickstarter projects and practice working with Google App Engine and AJAX. The backend uses BeautifulSoup to parse Kickstarter projects, and the frontend uses styles from Bootstrap. This project was inspired by Kickstarter, but is not officially affiliated in any way. All the code for this project is available on GitHub under an MIT license.
